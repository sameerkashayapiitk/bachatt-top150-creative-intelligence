# Bachatt — Top-150 Creative Intelligence

Interactive analysis of Bachatt's top 150 Meta ad creatives by spend (Oct 2025–Jun 2026),
decoded from transcripts + opening frames into **theme, emotion, hook and scroll-stopper**,
ranked by where spend actually went.

- **index.html** — emotional scoreboard (9 emotions by spend, CTR/CAC/stopping-power) + deep-dives on the top 3 (top-4 themes + top-10 ads each, with watch links & frame thumbnails).
- **explore.html** — filterable explorer across all 141 creatives (emotion / theme / hook / language + search).

Built from the analysis pipeline; static site, no backend.
